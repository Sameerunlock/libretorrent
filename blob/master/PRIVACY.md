Privacy Policy
=====================
Privacy Policy – TorrentBox

Last updated: 20 January 2026

TorrentBox (“the App”) is a torrent client application based on the open-source LibreTorrent project. This Privacy Policy explains how information is handled when you use the App.

By using TorrentBox, you agree to the terms described in this Privacy Policy.
Information Collection

### Information Collection

TorrentBox does not collect, store, or transmit any personal information to the developer or any external server.

Specifically:

No account creation

No name, email address, phone number, or location tracking

No behavioral analytics

No selling, sharing, or processing of user data

All torrent activity and file management occur locally on the user’s device.
## Permission and Access
TorrentBox requests permissions strictly required for core functionality:

### Storage / Manage All Files (Android 11 and above)

Used to:

Download torrent data

Read .torrent files

Manage downloaded files on the device

### Network Access

Used to:

Connect to torrent peers

Exchange data using peer-to-peer networking protocols

### Other Capabilities

View network status

Run at startup (optional)

Prevent device sleep during active downloads

Display notifications related to torrent status

None of these permissions are used to collect or transmit personal data.
## Types of Data Collected

### Data Storage
All torrents, downloaded files, and settings are stored only on the user’s device

The developer has no remote access to this data
### Usage Data
If the user voluntarily submits crash logs:

Logs may contain technical device information

No personal or identifying data is intentionally collected

Any email address provided is used only for troubleshooting

Logs are deleted once no longer required

Users may also report issues via the project repository.
